# The Coinforged

My first Slay the Spire mod. Still Actively Developing. 

Play as a runaway slot machine who uses cards that roll dice and spend poker chips. Increase your luck for a chance to get double rewards.

Planned Content:
- 80 new cards
- 4-5 new relics
- 3 new potions
